    Campaign Settings
    Generation Parameters
    Temperature0.80
    02

    How unpredictable the AI's word choices are. Low (0.2–0.5) = consistent and focused prose. Medium (0.7–1.0) = natural creative variation. High (1.2+) = wild and experimental — may become incoherent. Start here when tuning.

    Top-P0.95
    01

    Limits word choices to the most probable options that together make up this percentage of likelihood. 0.9–0.95 is a good default — it cuts off unlikely words while keeping variety. Lower values make responses more predictable; works alongside Temperature.

    Top-K0
    0 (off)200

    Hard limit on how many candidate words the model considers at each step. 0 = disabled (recommended — let Top-P handle this instead). If enabled, 40–100 is typical. Lower = safer but more repetitive; higher = more variety.

    Min-P0.05
    0 (off)1

    Cuts any word whose probability is less than this fraction of the top word's probability. Useful at high Temperature to remove truly improbable choices without killing variety. 0.05–0.1 works well alongside Temperature above 1.0. Leave at 0 if not using high Temperature.

    Repeat Penalty1.10
    1 (off)1.5

    Makes the AI less likely to reuse words it just used. 1.0 = no penalty. 1.1 is a gentle nudge that reduces word loops. 1.3+ strongly discourages repetition but can make prose feel forced. Raise this if the AI keeps repeating phrases.

    Max Tokens1024
    648192

    Maximum length of each AI response, measured in tokens (roughly ¾ of a word each). 512–1024 suits most roleplay exchanges. Raise to 2048+ for longer narrative scenes or detailed descriptions. Higher values use more VRAM and take longer to generate.

    Seed-1
    -1 (random)999 999 999

    -1 uses a random seed every response (recommended for normal play). Set a fixed number to make the AI produce the same output for the same input — useful for testing or recreating a specific response. Any number works; the value itself doesn't matter.

    Context Window32768
    4 096131 072

    Maximum number of tokens loaded into the AI's memory at once (context window size). Increase this if the AI loses track of earlier conversation — the default 32 768 handles roughly 50–80 turns. Must not exceed your model's actual context limit (check the model card). Higher values use significantly more VRAM.
